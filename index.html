Installation

  1. Open Revolt Chat
  2. Click on settings icon (top right)
  3. Click "Appearance" in left menu
  4. Scroll down to "Advanced Settings"
  5. Click on the CSS text area
  6. Paste this code:
/* Anarcho-Communisme Theme */ /* General background */ body { background-color: #121212; color: #f0f0f0; font-family: 'Roboto', sans-serif; } /* Main container */ #app { background-color: #0d0d0d; border-left: 3px solid #ff0000; border-right: 3px solid #ff0000; } /* Sidebar */ .sidebar { background-color: #1a1a1a; border-right: 2px solid #ff0000; } .sidebar a { color: #fff; font-weight: bold; text-transform: uppercase; } .sidebar a:hover { color: #ff0000; background-color: #222222; } /* Chat area */ .chat-container { background-color: #0d0d0d; border-left: 1px solid #ff0000; border-right: 1px solid #ff0000; } /* Messages */ .message { background-color: #1a1a1a; border-left: 3px solid #ff0000; color: #f0f0f0; } .message:hover { background-color: #262626; } /* Usernames */ .username { color: #ff0000; font-weight: bold; } .username.admin { color: #fff; } /* Text input */ .input-container { background-color: #1a1a1a; border-top: 2px solid #ff0000; } .input-container input { background-color: #0d0d0d; color: #f0f0f0; border: 1px solid #ff0000; } .input-container input:focus { border-color: #f0f0f0; } /* Buttons */ button { background-color: #ff0000; color: #f0f0f0; border: none; padding: 10px 15px; text-transform: uppercase; font-weight: bold; } button:hover { background-color: #f0f0f0; color: #ff0000; } /* Notifications */ .notification { background-color: #ff0000; color: #000; border-left: 4px solid #fff; } /* Links */ a { color: #ff0000; text-decoration: none; } a:hover { color: #f0f0f0; text-decoration: underline; }

Press "ESC" key at the top and enjoy your new theme!
